# CR Triage Checklist

- Is the claim reproducible in current HEAD?
- Is it correctness/platform/runtime or only style?
- Is there existing test coverage for this path?
- Does the fix introduce side effects in adjacent code paths?
- Did we verify with targeted tests?
- Did we post evidence (commit, files, test results) in thread reply?
- Should the thread be resolved now, or kept open with a concrete plan?
